Guía de Configuración
Sigue estos pasos una sola vez para conectar todo el sistema
1

Crear el Google Sheet

Ve a sheets.google.com, crea un nuevo spreadsheet y nómbralo "Plan Financiero Lucas & Josefa". No necesitas crear las hojas manualmente, el script las crea automáticamente.

2

Agregar el Apps Script

En el Google Sheet ve a Extensiones → Apps Script. Borra el código que hay y pega el contenido del archivo apps-script.js que descargaste. Guarda con Ctrl+S.

3

Ejecutar el Setup inicial

En Apps Script, selecciona la función setupSheets en el menú desplegable y haz clic en ▶ Ejecutar. Esto crea las hojas Gastos, Metas y Config automáticamente con todos los encabezados.

4

Publicar como Web App

En Apps Script haz clic en Implementar → Nueva implementación. Selecciona tipo "Aplicación web". En "¿Quién tiene acceso?" elige "Cualquier persona". Haz clic en Implementar y copia la URL que aparece.

// La URL tendrá este formato:
https://script.google.com/macros/s/AKfy.../exec
5

Conectar el Dashboard

Pega esa URL en el banner amarillo en la parte superior de este dashboard y haz clic en Guardar y conectar. El dashboard comenzará a leer y escribir en tu Google Sheet en tiempo real.

6

Crear el Google Form

Ve a forms.google.com y crea un formulario con estos campos exactos para que se conecte al Sheet:

Pregunta 1: "Fecha"            → Tipo: Fecha
Pregunta 2: "Categoría"        → Tipo: Opción múltiple (lista de categorías)
Pregunta 3: "Monto"            → Tipo: Párrafo corto (solo números)
Pregunta 4: "Pagado por"       → Tipo: Opción múltiple (Lucas / Josefa / Ambos)
Pregunta 5: "Método de pago"  → Tipo: Opción múltiple
Pregunta 6: "Descripción"      → Tipo: Párrafo corto (opcional)

En el Form ve a Respuestas → Vincular a Sheet y selecciona tu spreadsheet existente. Las respuestas llegarán a una nueva hoja llamada "Respuestas de formulario 1".

7

Subir a GitHub Pages (URL fija compartida)

Para que tanto Lucas como Josefa accedan desde cualquier dispositivo:

1. Crea cuenta en github.com (gratis)
2. Crea repositorio nuevo → nómbralo "finanzas"
3. Sube el archivo index.html (este archivo)
4. Ve a Settings → Pages → Source: main branch
5. Tu URL quedará: https://TU-USUARIO.github.io/finanzas
6. Comparte esa URL con Josefa por WhatsApp ✓
💡
¿Ya tienes todo listo? Pega la URL del Apps Script en el banner superior y el dashboard se conecta automáticamente. La URL se guarda en tu navegador, no necesitas ingresarla de nuevo.
