1

Reemplazar el Apps Script

En tu Google Sheet ve a Extensiones → Apps Script. Borra todo el código anterior y pega el contenido del nuevo archivo apps-script.js (v2). Guarda con Ctrl+S.

2

Ejecutar setupSheets de nuevo

Selecciona setupSheets en el menú y ejecuta ▶. Esto recrea las hojas con la estructura correcta (no borra datos existentes si las hojas ya tienen filas).

3

Nueva implementación

Ve a Implementar → Nueva implementación (no editar la anterior, crear una nueva).
· Tipo: Aplicación web
· Ejecutar como: Yo
· Quién puede acceder: Cualquier persona (sin "con cuenta de Google")
Copia la nueva URL generada.

4

Probar la URL directamente

Antes de pegar en el dashboard, abre la URL en el navegador añadiendo ?action=getAll al final. Deberías ver texto JSON. Si ves JSON, la conexión funcionará.

// Ejemplo de prueba en el navegador:
https://script.google.com/macros/s/TU-ID/exec?action=getAll

// Respuesta esperada (texto JSON):
{"gastos":[],"metas":[...],"ingresos":{...},"ok":true}
5

Conectar el Dashboard

Pega la URL base (sin parámetros) en el banner amarillo de arriba y haz clic en Conectar. El indicador verde confirmará la conexión.

6

Google Form → Sheets (ingreso desde celular)

Crea el Form en forms.google.com con estos campos:

Fecha          → Tipo: Fecha
Categoría      → Tipo: Lista desplegable (todas las categorías)
Monto          → Tipo: Respuesta corta
Pagado por     → Tipo: Opción múltiple (Lucas / Josefa / Ambos)
Método         → Tipo: Opción múltiple
Descripción    → Tipo: Respuesta corta (opcional)

En el Form: Respuestas → Vincular a spreadsheet → selecciona tu archivo de Google Sheets existente.

⚠️ Las respuestas irán a una hoja llamada "Respuestas de formulario 1". Para que el dashboard las lea, ve a esa hoja, selecciona los datos y cópialos a la hoja "Gastos" o usa un trigger automático.

7

Trigger: Form → Gastos automático

Para que las respuestas del Form lleguen automáticamente a la hoja Gastos, agrega esta función al Apps Script y crea un trigger "Al enviar formulario":

// Agrega esto al final del Apps Script:
function onFormSubmit(e) {
  const ss    = SpreadsheetApp.getActiveSpreadsheet();
  const sheet = ss.getSheetByName("Gastos");
  const r     = e.namedValues;
  const now   = new Date();
  sheet.appendRow([
    Utilities.formatDate(now,"America/Santiago","yyyy-MM-dd HH:mm:ss"),
    r["Fecha"]?.[0] || "",
    "Marzo 2026",
    r["Categoría"]?.[0] || "",
    "",
    0,
    Number(r["Monto"]?.[0]) || 0,
    r["Pagado por"]?.[0] || "",
    r["Método"]?.[0] || "",
    r["Descripción"]?.[0] || "",
  ]);
}

Luego en Apps Script: Triggers (ícono reloj) → Agregar trigger → onFormSubmit → Al enviar formulario.
